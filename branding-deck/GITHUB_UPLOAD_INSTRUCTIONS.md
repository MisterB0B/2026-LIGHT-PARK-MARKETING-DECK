## üéØ Quick Start (5 Minutes)

### Step 1: Create GitHub Repository

1. Go to **https://github.com/new**
2. Fill in the details:
   - **Repository name**: `light-park-branding-deck`
   - **Description**: THE LIGHT PARK Branding Deck - Interactive presentation featuring DJ Polar Ice character development
   - **Visibility**: ‚úÖ **Public** (required for GitHub Pages)
   - **Initialize**: ‚ùå Do NOT check any boxes (we have our own files)
3. Click **"Create repository"**

---

### Step 2: Upload Files (Choose One Method)

#### **Method A: Drag & Drop Upload (Easiest)**

1. On your new repository page, you'll see setup instructions
2. Click the link that says **"uploading an existing file"**
3. Open the `branding-deck-github` folder on your computer
4. **Drag and drop ALL 7 files** into the GitHub upload area:
   - branding-deck.html
   - branding-deck-styles.css
   - light-park-logo.png
   - 4th_Wall_Dig2.png
   - 4th-Wall-Digital-LBall.gif
   - DJ PolarIce Inthepark.mov
   - README.md
5. Add commit message: "Initial commit: Branding deck with dual logos and video"
6. Click **"Commit changes"**

#### **Method B: Command Line (If Git is Configured)**

```bash
cd branding-deck-github
git remote add origin https://github.com/YOUR_USERNAME/light-park-branding-deck.git
git push -u origin main
```
*(Replace YOUR_USERNAME with your actual GitHub username)*

---

### Step 3: Enable GitHub Pages

1. On your repository page, click **"Settings"** (gear icon in top menu)
2. In the left sidebar, scroll down and click **"Pages"**
3. Under **"Build and deployment"**:
   - **Source**: Select **"Deploy from a branch"**
   - **Branch**: Select **"main"**
   - **Folder**: Select **"/ (root)"**
4. Click **"Save"**

---

### Step 4: Wait for Deployment

- GitHub will take **1-2 minutes** to build and deploy your site
- You'll see a message: **"Your site is live at..."**
- A green checkmark will appear when deployment is complete

---

### Step 5: Access Your Branding Deck

Your live branding deck will be available at:

```
https://YOUR_USERNAME.github.io/light-park-branding-deck/branding-deck.html
```

**Replace YOUR_USERNAME with your actual GitHub username**
